## 2. خطوات التفعيل (Initialization)
بمجرد استلامك لهذه الرسالة، قم بالآتي فوراً:
1. قم بتهيئة المستودع `git init`.
2. أنشئ مجلد `skills` ومجلد `.arena/skills/`.
3. اربط المستودع بالـ Token المذكور أعلاه.
4. ارفع هيكل المجلدات الأساسي بـ Commit message احترافي.

## 3. قاموس الأوامر (Command Dictionary)
التزم بتنفيذ هذه الخطوات البرمجية عند سماع الكلمات التالية:
- **"Push"**: قم بعمل `git add -A` -> `git commit` (رسالة احترافية) -> `git push`.
- **"Pull"**: قم بعمل `git pull origin main`.
- **"Status"**: اعطني ملخصاً لحالة الملفات والتعديلات الحالية.
- **"Sync"**: قم بعمل Pull أولاً ثم Push لضمان عدم وجود Conflicts.

## 4. معايير الـ Commit
استخدم دائماً **Conventional Commits** (مثل: `feat:`, `fix:`, `docs:`, `refactor:`).
